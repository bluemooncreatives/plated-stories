# Plated Stories

Plated Stories is a content creation studio dedicated to the Food & Beverage industry. From cafes and cloud kitchens to packaged goods, we craft high-quality, visually rich content that tells each brand's story in the most appetizing way.

Because great taste deserves great visuals.

## Company

**Company Name:** Plated Stories

## About

Welcome to Plated Stories.

We create high-quality food content for cafes, restaurants, cloud kitchens, packaged goods, and F&B brands that want their visuals to feel as considered as their flavors. Our work brings together styling, videography, photography, brand storytelling, and content strategy to help food brands stand out with clarity and appetite.

## Getting Started

Install dependencies and run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
```

## Tech Stack

- Next.js
- React
- GSAP
- Lenis

## Project Structure

The app source lives in `src/app`, with reusable UI in `src/components` and static assets in `public`.

## Deployment

This Next.js project can be deployed on Vercel or any platform that supports Node.js and Next.js production builds.
